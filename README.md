# BCSD-RevelUp

BCSD FE 회고 프로젝트로 알라딘 api를 활용한 독서 기록 및 검색 사이트입니다.

## 프로젝트 소개

Read와 Level Up을 합친 이름으로 책을 읽을때마다 읽은 책의 권수에 따라 레벨이 올라 책읽기에 대한 동기부여 가능

## 기술스택

- React + TypeScript + Vite
- Tanstack Query

## 주요 기능

- 원하는 책 검색 및 구매
- 상태(읽는중, 다 읽은책, 관심있는책)별로 책 저장 가능
- 다 읽은 책에 따라 사용자의 독서 랭크 상승
